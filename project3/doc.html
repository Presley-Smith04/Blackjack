GOAL


My goal for this final project was to make an interactive and fun blackjack game
Creating this was not easy and it took a lot of time, and a few issues.
As of right now there are no console errors, and the only issue I am having
is that the second dealer card which is supposed to be hidden at the start, stays hidden and is
never physically shown, but it does still calculate the dealer total correctly.
